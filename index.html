Services

The services that I offer are:

Modelling

This includes hard-surface modelling, environment modelling anf a bit of sculpting. For hard-surface modelling, I mostly use manipulation of object volumes using their vertices, edges and faces. For environment modelling, which is my favourite, there are a number of methods I use. This includes use of displacement modifiers or the use of Blender's geometry nodes.

Surfacing/Texturing

For this, there are two methods that I use, namely procedural textures and image textures.

Lighting

This involves use of both HDRI lights for exterior lighting and inbuilt lighting systems such as sun lights, point lights, area lights and spot lights, and also irradiance volume lights for interior scenes

Animation

This is mostly done using keyframe animation.

Compositing

This means putting together different rendered layers and also adjusting the lighting and contrast properties of a rendered layer(s).

Rendering

This is done using either Evee render engine or Cycles render engine. Evee gives faster renders but cycles provides better lighting calculations hence better quality renders.
